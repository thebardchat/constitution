⚖ The Nine Pillars

01
Faith First
No project violates Christian values
Every tool, feature, and decision is built in alignment with faith. If it conflicts, it doesn't ship.
02
Family Stability
No build ships at the cost of family
Tiffany, Gavin, Kai, Pierce, Jaxton, Ryker come before any deadline. Sessions can interrupt. State gets saved.
03
Sobriety Integrity
Every tool reinforces health, never undermines it
The ecosystem is wired around the sobriety counter. Tools that enable destruction don't exist here.
04
Local-First AI
Pi before cloud. Privacy before convenience.
Raspberry Pi 5, local Ollama, local Weaviate. Data stays on-device unless there is explicit reason to leave.
05
80/20 Shipping
Done and deployed beats perfect and stalled
Ship the 80%. The last 20% of polish is often invisible to users and deadly to momentum.
06
Serve the Left-Behind User
Build for the ~800M Big Tech forgot
Windows 10 users losing support. People without $20/month for SaaS. The person Big Tech decided wasn't worth it.
07
Open by Default
Free when it can be. Paid only to fund the mission.
GPL v3 on everything that can be open. Paywalls only when they fund continued service for everyone else.
08
ADHD-Aware Design
Built for my brain first
Dashboards, briefings, and tools designed for the way ADHD actually works — not the way productivity gurus say it should.
09
Gratitude is Infrastructure
Credit Claude, Raspberry Pi, and Pironman — always
Collaborators get credited. Claude built half this. Raspberry Pi Foundation and Sunfounder enabled it. Say so.
